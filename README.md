# Agentic-Dev-Bootcamp (AI 编程智能体特训营)

欢迎来到 **Agentic-Dev-Bootcamp** 快速入门仓库！

本仓库旨在帮助研发、测试以及产品团队，快速跨越“网页聊天复制粘贴”的初级阶段，全面掌握基于终端与智能体（Agentic）架构的下一代 AI 辅助编程与自动化工程流。

## 🎯 核心目标

我们不再仅仅是“让 AI 写代码”，而是要学会**如何作为统帅与主架构师，指挥一支由 AI 组成的极其敏捷的开发兵团**。本仓库汇总了目前业界最顶尖的代码工具、MCP 协议生态以及经过实战检验的高效 Workflow。

## 📚 快速入门基石文档

请所有成员入营后，务必首先详细阅读我们的核心实战手册：
👉 [**《AI代码编辑工具使用培训材料.md》**](./AI代码编辑工具使用培训材料.md)

### 该培训手册为您深度拆解了以下核心板块：

1. **顶流双雄实战**：彻底打通全球顶尖嫡系产品 **Claude Code** 与 **Codex CLI** 的日常起手式，深入了解它们各自的“致命长板”与“避坑弱点”。
2. **给 AI 插上外部触角**：
   - 彻底搞懂 **MCP (Model Context Protocol)**：赋予模型读取文件、检索浏览器、操作 GitHub 的“物理外设”。
   - 掌握 **Skills**生态：不再写死板的 Prompt，利用 `SKILL.md` 为不同项目挂载专属的业务大脑。
3. **抵御幻觉与跑偏的最后防线**：
   - 极其关键的 `/plan` 谋定而后动模式：大需求必须先出图纸再写代码！
   - 通过 **OpenSpec** 驱动规范开发：人和 AI 共同确认系统边界。
4. **高阶全产研生态神器**：
   - 🔥 硅谷工业级流水线：挂载 **[gstack](https://github.com/garrytan/gstack)** 体验包含审查、测试及发布的严苛研发流。
   - 💰 账单与上下文拯救者：配置 **RTK (Rust Token Killer)** 过滤噪音。
   - 🎨 产品与 UI 的降维打击：接驳 **[Pencil MCP](https://pencil.dev/)**，实现“只讲需求线框图 -> 自动切出前端生产框架”的神级闭环。

## 🛠️ 改变习惯：拥抱终端原生 (CLI)

放下沉重的传统套壳 IDE 插件和网页对话框。基于 Terminal 原生的代码助手代表了最纯正的自动驾驶模式：
- 它们能自主读取你的报错环境、顺藤摸瓜检索并阅读全库上下文。
- 它们能直接替你敲击命令行、自动挂载测试、拉起构建甚至自主查日志修 Bug。
